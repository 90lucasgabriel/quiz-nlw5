## 5. Features
- Material design;
- Firebase;

  - Authentication with Google; 
- Anonymous answers are not allowed;

## 6. Installing and Running
- Clone this repo:
```
git clone git@github.com:90lucasgabriel/quiz-nlw5.git quiz
```

- Access directory:
```
cd quiz
```

- Run
```
flutter run
```

- Check browser or connected device;
